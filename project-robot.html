Proyecto: Un Robot

Este capitulo es sobre un ejemplo practico que describe como programar un robot para hacer entegas dentro de un peque√±o poblado que cuenta con 11 puntos de entrega.

El primer paso es hacer un array de los caminos existentes entre dos puntos de entrega adyacentes, que se expresa con el nombre de ambos lugares, separados por un guion.

El segundo paso es construir un objeto que contenga las "aristas" del poblado, colocando a cada lugar como nombre de propiedad, y como valor un array que incluya los lugares que colindan con el primero.

El tercer paso es hacer una funcion que basado en la posicion actual del robot, le indique a este ultimo a que lugar de entrega dirigirce a continuacion, y cada que llegue a un punto recalcular en base a los puntos de entrega restantes. Esto se hace mediante una clase, cual constructor recibe el lugar actual y un array de los paquetes restantes como argumentos.

Despues se hace una funcion que simule las acciones del robot y que le marque cuando ha acabado el trabajo (done), que es al entregar todos los paquetes

Despues de esto se empieza a utilizar diferentes metodos para decidir la secuencia de entregas, empezando por al azar. Despues el mail truck rout y despues el pathfinding
